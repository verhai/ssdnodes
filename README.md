## SSD Nodes 是什麼

SSD Nodes 是一家 Linux VPS 主機商，總部在加州 Tustin，從 2011 年開始做到現在，超過 40,000 台虛擬機跑過他們的平台。他們不搞花俏的 UI，不做大規模廣告，就是把成本壓到極低然後傳給用戶。

他們怎麼做到的？兩件事：

第一，他們自己開發了一套叫 **Vippy** 的虛擬化平台。別家主機商要花錢買第三方 hypervisor 授權，他們不用，這部分省下來的錢直接反映在你的帳單上。

第二，他們直接跟 Hivelocity、Hurricane Electric、GTT、Cogent 這些大型機房合作，不透過中間商，基礎設施成本更低。

結果就是：同樣規格的機器，在 DigitalOcean 要 $96/月，在這裡可能 $9 出頭。
